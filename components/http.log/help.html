HTTP服务实时日志 用法

  1. 假设 HTTP服务实时日志 服务地址是 http://192.168.1.20:8000/http.log.
  2. 浏览器打开 http://192.168.1.20:8000/http.log 查看日志。
  3. 在你的程序中，将日志数据发送到 http://192.168.1.20:8000/http.log/log. 此接口的文档见下方说明

log 的接口文档:

        
请求方式：POST

请求参数：

```
{
	client:       // 客户端标识，                          字符串
	timeStamp:    // 时间戳，传UNIX TIMESTAMP 毫秒数，      整数
	method:       // 请求方式，比如 "POST", "GET"，         字符串
	url:          // 请求地址，                            字符串
	params:       // 请求参数，                            字符串
	statusCode:   // response status code, 比如 404, 200, 整数
	responseBody: // 返回内容，                            字符串
}
```

返回值："success"
